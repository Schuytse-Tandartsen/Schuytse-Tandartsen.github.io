# Website

## Installation
Use ruby 3.0.1:
```
rbenv local 3.0.1
```

Install Gems:
```
bundle
```

## Development
Run with live reload:
```
bundle exec jekyll serve --livereload
```

## Pricing page

- The plugin is inside the \_plugins-folder.
- The config is inside the config.yml-file.
- The layout used is layouts/prices.html.
